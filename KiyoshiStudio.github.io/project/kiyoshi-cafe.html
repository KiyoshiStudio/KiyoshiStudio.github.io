安安靜靜，安農溪畔的咖啡廳

咖啡煎焙所 喜代志 是一間專注於咖啡烘焙的咖啡廳，提供：

  • 專業烘焙精品咖啡
  • 賽風虹吸咖啡
  • 經典風味甜品

喜老闆自己會烘咖啡、做一些好好吃的經典甜品，在咖啡煎焙所 喜代志的店裡，賽風壺、義式咖啡機是主要的咖啡萃取工具，你可以在咖啡煎焙所 喜代志的店裡，得到喜老闆對於咖啡體驗的基準。

接下來介紹的是，喜老闆想透過這樣的店，傳達的體驗基準：

世界各產區的精選、精品咖啡

店裡提供的咖啡品項不多，並不會眼花撩亂，對於只想要嚐鮮的咖啡玩家們，先說聲抱歉，趕流行本來就不是喜老闆的個性。

喜代志店裡的咖啡，選用能夠忠實反映產區風土，同時也有著優良品質的生產者，有時會提供稀有、頂級精品咖啡品項，盡量以能夠更實質幫助生產者的途徑採購生豆，讓提供價值的生產者們都能感到物有所值的美好。

賽風壺(Siphon) - 自我要求的體現

曾經有客人這樣問喜老闆：

「喜老闆，你主張手沖比較方便，又比較容易即刻修正、容易好喝，那賽風的存在還有意義嗎？」

使用賽風，是咖啡從業人員自我要求的終極體現，計較到最後，當然是對來店裡消費的客人最有利。

素養好的師傅，使用賽風壺萃取的咖啡，完完全全反應所使用咖啡豆的選豆、烘焙、品管、沖煮等各方面的品質，以及當下豆子的狀態，如果要真的絕對好喝，早在沖煮之前就需要許多關鍵步驟的自我要求，才能成就每一杯好喝的咖啡。

這樣的一杯咖啡內含的，超出「使用好原料的自家烘焙」多得太多了。

物有所值，才能讓世界裡付出的許多價值，都能在極少的壓迫之下得到適當回報:也包含了你我上班、執業的付出。

親手製作的經典風味甜品

店裡的經典風味甜品，都是喜老闆親手製作。
